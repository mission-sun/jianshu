# 简书项目

仿照简书网站搭建，技术站：React + React Router  + React Redux + Redux + styled-components

## npm script 命令

``` bash
# 安装依赖
npm install

# 开启本地server
npm run dev

# 构建生产环境代码，打包生成可以直接上传通天塔的zip包
npm run build

# 构建生产环境代码，并生成可视化的bundle分析报告
npm run build --report
```

## 功能实现

 1. 轮播图实现    react-slick
 2. 样式组件化    styled-commponents
